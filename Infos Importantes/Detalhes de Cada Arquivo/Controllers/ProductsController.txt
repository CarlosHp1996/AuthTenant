ProductsController.cs - Controlador de Produtos
Função Principal: CRUD completo para gerenciamento de produtos com funcionalidades avançadas.

Endpoints Disponíveis:

GET /api/products
Lista produtos com paginação e filtros opcionais
Parâmetros: page, pageSize, searchTerm, isActive
Validação de parâmetros de paginação (1-100 items por página)

GET /api/products/{id}
Busca produto específico por ID (GUID)
Validação de GUID não-vazio

POST /api/products
Cria novo produto
Retorna status 201 Created com location header
Validação completa de dados de entrada

PUT /api/products/{id}
Atualiza produto existente
Validação de ID e dados de entrada

DELETE /api/products/{id}
Soft delete (marca como deletado, preserva dados)
Retorna confirmação com ID do produto

GET /api/products/active
Endpoint especializado para produtos ativos
Atalho que chama o endpoint principal com filtro isActive=true

Características Técnicas:
Autorização obrigatória ([Authorize]) em todos os endpoints
Validação rigorosa de parâmetros (GUIDs, paginação, null checks)
Documentação OpenAPI detalhada com todos os status codes possíveis
Tratamento diferenciado para operações de criação (201 vs 200)
Suporte a busca textual (nome, descrição, SKU)
Constraint de rota para GUIDs ({id:guid})
