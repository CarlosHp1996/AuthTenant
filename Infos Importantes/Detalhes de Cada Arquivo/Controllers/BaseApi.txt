BaseApiController.cs - Controlador Base Abstrato
Função Principal: Classe base abstrata que fornece funcionalidades comuns para todos os controllers da API.

Responsabilidades:
Injeção de Dependência: Gerencia a instância do MediatR para padrão CQRS
Tratamento de Respostas: Dois métodos principais para conversão de Result<T> em ActionResult
HandleResult<T>() - Retorna ActionResult<T> genérico
HandleResultAsActionResult<T>() - Retorna ActionResult não-genérico
Mapeamento de Status HTTP: Lógica inteligente que determina códigos de status baseado no conteúdo do erro
Padronização de Respostas: Estrutura consistente para respostas de sucesso e erro
Métodos Utilitários:
ValidationError() - Para erros de validação estruturados
SuccessResponse() - Para respostas de sucesso padronizadas
Timestamp Automático: Adiciona timestamp UTC em todas as respostas de erro

Características Técnicas:
Suporte multilíngue (português/inglês) no mapeamento de erros
Análise semântica de mensagens de erro para determinar status codes apropriados
Documentação OpenAPI/Swagger automática para responses comuns
